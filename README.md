# Quote of Day Generator
> Single page QOTD generator using the [Zen Quotes](zenquotes.io) API

- Deployed on [AWS Amplify](https://aws.amazon.com/amplify/) and uses AWS [Lambda](https://aws.amazon.com/lambda/) for the API call. 
- Used [Gemini](https://gemini.google.com/) AI to create the pages and code.
